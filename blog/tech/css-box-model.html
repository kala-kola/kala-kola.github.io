The CSS Box Model

Layers of the Box Model

The CSS Box model is used for design and layout. It contains layers which wrap around every HTML element. The outer most layer is the margin, is the area outside the border and pushes boxes away from boxes. Next is the border that goes around the padding and content. Padding comes next and pushes content further inside the box.

Box Model Example

In the screen shot above, the margin is white around the outside of the box, the border is purple and the padding is the space around the content in the pink area.

Width and Height of an Element

All these layers means the TOTAL WIDTH of the element is not only the content width but the size of the padding on either side plus the border on either side and the margin on both sides as well.
Just like you layer up in winter, you want to make sure your outer shell fits all the inside layers.
TOTAL HEIGHT is calculated the same way: Height of content + padding top and bottom + border top and bottom + padding top and bottom.

Order of comand in CSS

When you use CSS to indicate the margin or padding the order begings at the Top and proceeds Right, Bottom and Left. Take the example in the screen shot left.
